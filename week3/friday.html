Example code

XHR with Content-Type and responseType:

var xhr = new XMLHttpRequest;
xhr.open('post', 'http://galvanize-node-red-demo.mybluemix.net/foo');
xhr.setRequestHeader('Content-Type', 'application/json');
xhr.responseType = 'json';
xhr.addEventListener('load', function () {
  console.log(xhr.response);
});
xhr.send(JSON.stringify({hello: "nodered"}));

CORS configuration for NodeRed:

    httpNodeCors: {
      origin: "*",
      methods: "GET,PUT,POST,DELETE"
    },
