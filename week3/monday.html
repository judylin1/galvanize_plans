Absolute Paths

Add valid HTML structure (doctype, head, body etc...) to each index.html file and deploy to S3.

Using absolute paths (including file://... do the following:

  • Link all 3 css files successfully to all three index.html files (add a few divs to each page and style those divs from the different css files to make sure that you have correctly linked each CSS file)
  • Link the 2 javascript files to all three index.html documents - add some console.log statements to make sure you've done it correctly

Deploy to S3 again. What happened?

Relative Paths

In all .html files, convert all paths to relative paths. Make sure it works locally, then deploy to S3 and make sure it works there.

Relative Paths

Change all paths to be root-relative (starts with a /). Note how it doesn't work locally. Why? What's the rule for a relative path?

That is, what is it actually relative to?

Running static sites with a web server

Install http-server - a simple HTTP server written in Node.

npm install http-server

Change all paths to be root-relative, open your site with http-server and then see the links work.
