Forms Exercise

Fork and clone the exercise repo. Work through each challenge one at a time, practicing swift developer workflow and window management.

https://github.com/gSchool/html-forms

  • Fork
  • Clone
  • Stage changes (git add -A)
  • Commit work regularly (git commit -m "description of your change")
  • Push to your repo (git push origin master)
  • Submit a pull request after pushing the first time from https://github.com
