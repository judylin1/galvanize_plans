Analyze this code (variable tables)

var i = 0;
var output = "";
while(i < 4) {
  var tmp;
  if(i % 2 === 0) {
    tmp = "zebra";
  } else {
    tmp = "stripe";
  }
  output += tmp;
  if (i < 3){
      output += "-";
  }
  i++;
}

For Loops

For loops are a concise way to express how a loop should be initialized, how long a loop should run, and what action should be taken at the end of each iteration. The general syntax is below:

for (initialization; conditional; final expression) {
    // Some code to run.
}
