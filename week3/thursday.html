Building a library

Introducing Mimetic.

Mimetic

Set up the repo

On Github, create a new repository named "mimetic"

  • Choose "Initialize w/ README"
  • Add .gitignore for Node
  • Add Apache License 2.0

Clone it locally

  • clone your new repository (with the SSH url)
  • cd into it
  • open it in atom

Make it an NPM package

npm init --yes

Type your name in when prompted and hit enter.

Install and setup jasmine with:

npm install -g jasmine
jasmine init
touch spec/querystring_spec.js

Add your first spec

Paste the following code into spec/querystring_spec.js

var parseQueryString = require('../lib/querystring');

describe('parseQueryString', function () {
  it('turns a querystring into an object', function () {

    // Part 1: setup your input
    var input = '?artist=Miles&year=1948';

    // Part 2: call your function,
    //    passing in the input
    //    and capturing the result in a variable named `actual`
    var actual = parseQueryString(input);

    // Part 3a: define your expected output for the given input
    var expected = { artist: 'Miles', year: '1948' }

    // Part 3b: check to see if your code produced the results you expected
    expect(actual).toEqual(expected);

  });
});
