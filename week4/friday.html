Install Mongo

brew update
brew install mongodb
ln -sfv /usr/local/opt/mongodb/*.plist ~/Library/LaunchAgents
launchctl load ~/Library/LaunchAgents/homebrew.mxcl.mongodb.plist

See more detailed instructions here: http://docs.mongodb.org/manual/tutorial/install-mongodb-on-os-x/

Learn Mongo

  • Overview: https://docs.mongodb.org/getting-started/shell/introduction/
  • Import Sample Data: https://docs.mongodb.org/getting-started/shell/import-data/
