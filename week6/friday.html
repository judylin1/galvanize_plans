Interacting with APIs in Node

Objectives:

By the end of this you should be able to:

  • Sequence diagram an express / API integration flow
  • Get data from a site that uses tokens in the URL using unirest
  • Get data from a site that uses headers in the URL using unirest

Guiding questions:

  • Why can't you use XMLHttpRequest in node?

Process

  • Register for an API key
  • Go to the docs
    • Figure out what to do
    • Use the online console / playground if available
  • Install unirest (be sure to save it to package.json)
  • require unirest and make the api call
  • Ensure that your key is not in Git
    • install the dotenv npm package (be sure to save it to package.json
    • create a .env file
    • add .env to your gitignore
    • require / load dotenv (probably in app.js)
    • add environment variables / values in .env
    • replace any hard-coded keys with process.env
  • Parse the data as needed and display / add to your database

Unirest

  • http://unirest.io/nodejs.html

Sending API Tokens in QueryStrings

  • http://developer.nytimes.com/io-docs
  • http://developer.nytimes.com/apps/mykeys
unirest.get('http://api.nytimes.com/svc/books/v3/lists/hardcover-fiction.json?api-key=5551212')
  .end(function (response) {
    console.log(response.body);
  })

Sending API Tokens in Headers

  • https://www.pivotaltracker.com/help/api#top
  • https://www.pivotaltracker.com/profile
unirest.get('https://www.pivotaltracker.com/services/v5/projects')
  .header('X-TrackerToken', '5551212')
  .end(function (response) {
    console.log(response.body);
  })
