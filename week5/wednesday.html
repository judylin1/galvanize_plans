Deploying to Heroku

  • Install the Heroku Toolbelt
  • cd into your Mail Merge project
  • make sure you are on the master branch, and that your code is on master
  • Run heroku apps:create
  • Run git push heroku master
  • When it's done, run heroku open

Deploying with Mongo

MongoLab is a free service for hobby Mongo databases. https://devcenter.heroku.com/articles/mongolab

To deploy you albums app:

  • cd into your albums project
  • make sure you are on master, and your code is on master
  • change your database connection line to
  var db = require('monk')(process.env.MONGOLAB_URI || 'localhost/album_class_demo');
  • git add and commit
  • heroku apps:create
  • git push heroku master
  • heroku addons:create mongolab
