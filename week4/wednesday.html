So Much to Diagram!!!

What does this algorithm do? On paper, draw the diagram of what the result object (and all related objects) look like.

Hints:

  • write your name on the paper
  • your diagram should only have 1 variable: result
  • start in the upper left-hand corner of the page for the final result
  • it may help to write out the variables root, parent, child for each iteration on a whiteboard (or spreadsheet, or separate sheet of paper)

Please do not run the code until you have turned in your diagram.

var linkedObjects = function (input) {
  var root, parent, child;
  for (var i = 0; i < input.length; i++) {
    child = {value: input[i], next: null};
    if(parent) {
      parent.next = child;
    }
    if(!root) {
      root = child;
    }
    parent = child;
  }
  return root;
};

var result = linkedObjects([1,23,68]);
